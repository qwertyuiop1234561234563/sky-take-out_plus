# sky-take-out_plus
主要使用了springboot＋ssm框架＋vue+redis来进行开发的

1.添加了redission实现分布式锁去防止用户重复下单的问题，还有解决缓存击穿的问题
2.添加了线程池来进行优化，异步处理非核心内容，去优化用户的体验
3.给mysql中高访问的数据添加了联合索引加快了查询的速度
4.解决了缓存穿透的问题（缓存空对象），还有缓存雪崩的问题（随机存活时间）
5.用缓存注解对高访问的菜品和套餐进行缓存
6.通过websocket向前端传输数据，利用springtask实现超时取消订单
7.使用了jwt令牌，自定义拦截器进行拦截完成用户认证，通过treadlocal配合拦截器进行token验证
